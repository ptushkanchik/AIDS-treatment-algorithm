Программа представляет собой интерактивный алгоритм лечения ВИЧ-инфицированных пациентов: выбор антиретровирусной терапии нативным пациентам, назначение профилактики передачи ВИЧ от беременных женщин плоду, лечение оппортунистических инфекций. В алгоритме использованы материалы клинических протоколов, разработанных Министерством здравоохранения Украины.
